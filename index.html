Custom Flex Grid

Como usar

Primeiro, para criar as grids, chame a função mixins para criar as grids.
Em Mixins, existem 4 função, xs-grid, sm-grid, md-grid, lg-grid, que cria as classes, com seu próprio prefixo. No case eu estou criando dentro de breakpointers diferentes para respeitarem as resoluções.

.grid {
	xs-grid(12);
	display: flex;
	flex-wrap: wrap;
	margin: 0 -15px;
	> * {
		width: 100%;
		padding: 0 15px;
	}
	* {
		box-sizing: border-box;
	}
}

@media screen and (min-width: sm) {
	.grid {
		sm-grid(12);
	}
}

@media screen and (min-width: md) {
	.grid {
		md-grid(12);
	}
}

@media screen and (min-width: lg) {
	.grid {
		lg-grid(12);
	}
}

A seguir, no seu HTML, chame a classe .grid e dentro dela coloca as grids, com o tamanho relativo a função que você passou.

<div class="grid">
	<div class="sm-6-6"></div>
	<div class="sm-3-6"></div>
	<div class="sm-3-6"></div>
</div>
